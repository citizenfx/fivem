## DELETE_COLSHAPE
```c
bool DELETE_COLSHAPE(char* colShapeId);
```
Deletes a colshape by identifier
## Parameters
* **colShapeId**: The Identifier of the colshape to be deleted.
## Return value
true if deleted successfully, false if it didn't exist in the first place
