## SET_LIGHT_EXTRAFLAGS

```c
bool SET_LIGHT_EXTRAFLAGS(int lightIndex, int extraFlags);
```

Set additional configuration flags for an existing light

## Parameters

* **lightIndex**: The index of the target light
* **extraFlags**: Bitmask of extra flags

## Return value
A boolean indicating whether the operation succeeded.
