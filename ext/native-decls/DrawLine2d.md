## DRAW_LINE_2D

```c
void DRAW_LINE_2D(float x1, float y1, float x2, float y2, float width, int r, int g, int b, int a);
```

Like DRAW_RECT, but it's a line.

## Parameters
* **x1**: First x.
* **y1**: First y.
* **x2**: Second x.
* **y2**: Second y.
* **width**: Width of line.
* **r**: Red.
* **g**: Green.
* **b**: Blue.
* **a**: Alpha.
