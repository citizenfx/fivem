## SET_LIGHT_VOLUMETRIC_FADE_DISTANCE

```c
bool SET_LIGHT_VOLUMETRIC_FADE_DISTANCE(int lightIndex, int volumetricFadeDistance);
```

Set the fade distance for volumetric lightingn.

## Parameters

* **lightIndex**: The index of the created light
* **volumetricFadeDistance**: The maximum distance

## Return value
A boolean indicating whether the operation succeeded.
