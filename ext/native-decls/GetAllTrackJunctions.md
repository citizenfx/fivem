## GET_ALL_TRACK_JUNCTIONS

```c
object GET_ALL_TRACK_JUNCTIONS();
```

Returns all track junctions on the client
The data returned adheres to the following structure:
```
[1, 2, 4, 6, 69, 420]
```

## Return value
An object containing a list of track junctions ids.
