## REGISTER_FONT_FILE

```c
void REGISTER_FONT_FILE(char* fileName);
```

Registers a specified .gfx file as GFx font library.
The .gfx file has to be registered with the streamer already.

## Parameters
* **fileName**: The name of the .gfx file, without extension.
