## SET_LIGHT_COORDS

```c
bool SET_LIGHT_COORDS(int lightIndex, float x, float y, float z);
```

Set the world coordinates of a specified light.

## Parameters

* **lightIndex**: The index of the light
* **x**: The X coordinate
* **y**: The Y coordinate
* **z**: The Z coordinate

## Return value
A boolean indicating whether the operation succeeded.
