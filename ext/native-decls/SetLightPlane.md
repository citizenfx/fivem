## SET_LIGHT_PLANE

```c
void SET_LIGHT_PLANE(float x, float y, float z, float w);
```

Set the plane parameters for a light.

## Parameters

* **x**: X component of the plane
* **y**: Y component of the plane
* **z**: Z component of the plane
* **w**: W component of the plane
