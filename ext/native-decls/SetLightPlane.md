## SET_LIGHT_PLANE

```c
bool SET_LIGHT_PLANE(int lightIndex, float x, float y, float z, float w);
```

Set the plane parameters for a light.

## Parameters

* **lightIndex**: The index of the created light
* **x**: X component of the plane
* **y**: Y component of the plane
* **z**: Z component of the plane
* **w**: W component of the plane

## Return value
A boolean indicating whether the operation succeeded.
