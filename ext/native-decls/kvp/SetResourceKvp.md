## SET_RESOURCE_KVP

```c
void SET_RESOURCE_KVP(char* key, char* value);
```

A setter for [GET_RESOURCE_KVP_STRING](#_0x5240DA5A).

## Parameters
* **key**: The key to set
* **value**: The value to write

