## IS_PLAYER_COMMERCE_INFO_LOADED

```c
BOOL IS_PLAYER_COMMERCE_INFO_LOADED(char* playerSrc);
```

> This native is deprecated and may be removed in a future version. Use [`IS_PLAYER_COMMERCE_INFO_LOADED_EXT`](#_0x1D14F4FE) instead.

Requests whether or not the commerce data for the specified player has loaded.

## Parameters
* **playerSrc**: The player handle

## Return value
A boolean.
